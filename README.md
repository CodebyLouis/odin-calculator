## About

This is a modern browser-based calculator with a sleek design and dynamic color theming. Pick any color you want and watch the entire interface adapt with harmonious color variations.

## What I Built

- Full calculator functionality (add, subtract, multiply, divide)
- Random color theme generator 
- Dynamic CSS color system using `color-mix()`
- Smooth transitions and hover effects
- GitHub link integration
- Decimal rounding for clean display

## Technologies

- HTML
- CSS (Flexbox, CSS variables, color-mix, transitions)
- JavaScript (DOM manipulation, event handling)

## Features

- Chain calculations (results carry over to next operation)
- Adaptive color theming from a single primary color
- Neumorphic button design with glassmorphism
- Responsive display with overflow handling
